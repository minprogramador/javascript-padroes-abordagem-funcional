

## Projeto
### Buscando notas fiscais por API com js.

## Requesitos.

* Node.js 8.0 ou superior
* Google Chrome versão 63 ou superior.

---

## Subindo o servidor
*Entre na pasta do projeto e execute.*
```
node server
```

*Se tudo der certo!*
```
Server is running at http://localhost:3000
```
